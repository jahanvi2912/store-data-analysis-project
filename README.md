# 📊 Store Data Analysis Project

## 📌 Overview
This project focuses on analyzing store-related data to identify patterns, trends, and relationships between various factors such as temperature, fuel price, CPI, and unemployment.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Key Analysis
- Data Cleaning & Preprocessing
- Distribution Analysis
- Correlation Heatmap
- Basic Machine Learning (Linear Regression)

## 🔍 Insights
- Most variables show weak correlation with each other
- Data distribution highlights consistent patterns across observations
- External factors collectively influence store conditions

## 🤖 Model
A simple Linear Regression model was used to understand relationships between variables and was evaluated using Mean Squared Error (MSE).
